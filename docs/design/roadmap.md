# Roadmap

Future development plans for Flux.

## Current Status: v0.1

- [x] Core training loop
- [x] Adaptive async controller
- [x] GRPO, PPO, DPO, REINFORCE algorithms
- [x] SGLang integration
- [x] CUDA IPC weight sync
- [x] Basic checkpointing

## v0.2 (Planned)

- [ ] Multi-node training
- [ ] vLLM backend support
- [ ] Advanced curriculum learning
- [ ] Reward model training
- [ ] RLHF evaluation suite

## v0.3 (Future)

- [ ] Online DPO
- [ ] Rejection sampling fine-tuning
- [ ] Constitutional AI support
- [ ] Automated hyperparameter tuning

## v1.0 (Long-term)

- [ ] Production-ready stability
- [ ] Comprehensive documentation
- [ ] Large-scale benchmarks
- [ ] Enterprise features
